# Path Tracing 算法

+ 首先从相机处，随机采样一个光线r，求出与场景的交点p，希望得到p处着色的信息
+ 如果没有交点，返回空
+ 如果p是光源，直接返回光照强度
+ 否则分直接光照和间接光照计算
+ 如果p是表面的点，就对于光源随机采样，得到x，然后计算p和x的法线，p指向x的方向，p与x的距离，x处亮度
+ 如果p与x之间被遮挡，直接光照为0，否则计算直接光照
+ 再考虑间接光照，如果通过轮盘赌，在p处随机采样出射光w，计算交点q
+ 如果q不是光源（因为避免重复计算直接光照），对于q递归计算q对于w方向的光线的贡献
+ 递归层数期望可由轮盘赌的概率计算出
